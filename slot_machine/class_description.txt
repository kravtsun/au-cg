Общее описание классов

GLUTWindowManager -> WindowManager - классы, обеспечивающие инициализацию OpenGL и оконного контекста (а также базовые методы, характерные для оконного окружения)

GLHolder - некоторый объект, который отвечает за отрисовку всего предметного контекста.

Button - Некий элемент, задуманный для отрисовки в заданном месте экрана и с заданными размерами.

AbstractPass - некоторая сущность, которая может что-то отображать в контексте заданном извне метода pass, может менять состояние от одного вызова pass (предполагается, что вызовы будут происходить по одному на каждый кадр) к другому.

StartButtonPass - Pass, отображающий кнопку старта в конкретном месте экрана.

CellHolder - абстракция, служащая для сбора информации о состоянии одной ячейки игрового автомата

CellPass - управляющий Pass для нескольких ячеек автомата.

Wrappers/* -  обёртки над сущностями OpenGL, в основном, для уверенности в освобождении ресурсов.

Roll/* - сущности, включающие скорости поворота ячеек. Поведение скоростей можно настраивать через эти классы.

Color/* - абстракции цвета, поведение которого можно настраивать по ходу смены фреймов.
